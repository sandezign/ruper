# Rumper Marketing Website

Area untuk landing page Bahasa Indonesia berbasis Next.js dan Tailwind CSS.

## Scope Awal

- Menjelaskan masalah dan value proposition Rumper.
- Menjelaskan paket dan harga.
- Menampilkan contoh isi laporan dan FAQ.
- Mengarahkan CTA utama ke Typeform melalui `NEXT_PUBLIC_TYPEFORM_URL`.
