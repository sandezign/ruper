# CHK-RPT-001: Checklist Persetujuan Final Report oleh Admin

Gunakan checklist ini untuk setiap Mini Check, Full Report, dan Compare Report, termasuk report yang tidak melibatkan analyst.

## A. Identitas dan Versi

- [ ] Report ID tersedia dan unik.
- [ ] Report type sesuai kebutuhan customer.
- [ ] Template Version ID aktif atau masih didukung.
- [ ] Document version mengikuti aturan versi.
- [ ] Property atau lokasi dapat diidentifikasi.

## B. Consent, Privacy, dan Akses

- [ ] Consent pemrosesan data telah diberikan.
- [ ] Disclaimer telah ditampilkan dan diakui.
- [ ] Report tidak memuat data pribadi yang tidak diperlukan.
- [ ] Hak akses internal sesuai kebutuhan kerja.
- [ ] Share link, bila ada, menggunakan view-only secara default.
- [ ] Tidak ada data customer yang disimpan dalam repository Git.

## C. Kelengkapan Minimum

- [ ] Tujuan evaluasi jelas.
- [ ] Minimal satu finding tersedia.
- [ ] Minimal satu risk tersedia.
- [ ] Minimal satu recommendation tersedia.
- [ ] Minimal satu next step dan owner tersedia.
- [ ] Overall status dan decision required tersedia.

## D. Evidence dan Confidence

- [ ] Lima kategori wajib telah diperiksa: banjir, commute, akses fisik, fasilitas esensial, dan kualitas lingkungan/red flags.
- [ ] Skor kategori, bobot, dan skor total mengikuti `STD-SCR-001`.
- [ ] Finding penting memiliki evidence atau alasan evidence belum tersedia.
- [ ] Source dan tanggal observasi tersedia bila relevan.
- [ ] Confidence label sesuai dengan kekuatan evidence.
- [ ] Evidence yang konflik dijelaskan.
- [ ] Batasan data dan kebutuhan validasi lapangan terlihat.
- [ ] Critical red-flag override telah diterapkan bila relevan.

## E. Bahasa dan Keamanan Klaim

- [ ] Fakta, interpretasi, dan rekomendasi dapat dibedakan.
- [ ] Tidak ada klaim `pasti aman`, `bebas risiko`, `dijamin`, atau klaim absolut lain.
- [ ] Tidak ada jaminan investasi atau prediksi keuntungan.
- [ ] Rekomendasi bersifat actionable dan proporsional.
- [ ] Disclaimer konsisten dengan isi report.

## F. Export dan Delivery

- [ ] Cover, isi, appendix, version stamp, dan export date tampil benar.
- [ ] Draft watermark tidak muncul pada final.
- [ ] Link dan evidence yang ditampilkan dapat dibuka sesuai izin.
- [ ] Final report telah dikunci dari perubahan langsung.
